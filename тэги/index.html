HTML-тэги: наглядная иллюстрация

TITLE: определяет заголовок документа. Элемент <title> не является частью документа и не показывается напрямую на веб-странице. В операционной системе Windows текст заголовка отображается в левом верхнем углу окна браузера. Допускается использовать только один тег <title>.
<title>HTML-тэги</title>

h1: заголовок первого уровня, блочный элемент, всегда с новой строки, следующие после него элементы отображаются с новой строки.

Заголовок 1-го уровня
<h1>Заголовок 1-го уровня</h1>

h2: заголовок второго уровня, блочный элемент, всегда с новой строки, следующие после него элементы отображаются с новой строки.

Заголовок 2-го уровня
<h2>Заголовок 2-го уровня</h2>

h3: заголовок третьего уровня, блочный элемент, всегда с новой строки, следующие после него элементы отображаются с новой строки.

Заголовок 3-го уровня
<h3>Заголовок 3-го уровня</h3>

h4: заголовок четвёртого уровня, блочный элемент, всегда с новой строки, следующие после него элементы отображаются с новой строки.

Заголовок 4-го уровня
<h4>Заголовок 4-го уровня</h4>

h5: заголовок пятого уровня, блочный элемент, всегда с новой строки, следующие после него элементы отображаются с новой строки.

Заголовок 5-го уровня
<h5>Заголовок 5-го уровня</h5>

h6: заголовок шестого уровня, блочный элемент, всегда с новой строки, следующие после него элементы отображаются с новой строки.

Заголовок 6-го уровня
<h6>Заголовок 6-го уровня</h6>

b: устанавливает жирное начертание шрифта. Допустимо использовать этот тег совместно с другими тегами, которые определяют начертание текста.

Жирный
<b>Жирный</b>

i: устанавливает курсивное начертание шрифта. Допустимо использовать этот тег совместно с другими тегами, которые определяют начертание текста.

Курсив
<i>Курсив</i>

u: добавляет подчеркивание к тексту. Этот тег осуждается спецификацией HTML, взамен рекомендуется использовать стили.

Подчёркивание
<u>Подчёркивание</u>

a: предназначен для создания ссылок. В зависимости от присутствия атрибутов name или href тег <a> устанавливает ссылку или якорь.

Ссылка на ya.ru
<a href="http://ya.ru">Ссылка на ya.ru</a>

big: увеличивает размер шрифта на единицу по сравнению с обычным текстом. В HTML размер шрифта измеряется в условных единицах от 1 до 7, средний размер текста, используемый по умолчанию, принят 3.

Большой текст
<big>Большой текст</big>

small: уменьшает размер шрифта на единицу по сравнению с обычным текстом. В HTML размер шрифта измеряется в условных единицах от 1 до 7, средний размер текста, используемый по умолчанию, принят 3.

Маленький текст
<small>Маленький текст</small>

strong: акцентирует текст, отображая его жирным начертанием.

Акцентированный текст
<strong>Акцентированный текст</strong>

sub: отображает шрифт в виде нижнего индекса. Текст при этом располагается ниже базовой линии остальных символов строки и уменьшенного размера..

Скорость угловая
Скорость<sub> угловая</sub>

sup: отображает шрифт в виде верхнего индекса. Текст при этом располагается выше базовой линии остальных символов строки и уменьшенного размера..

Скорость в квадрате
Скорость<sup> в квадрате</sup>

br: устанавливает перевод строки в том месте, где этот тег находится. В отличие от тега абзаца <p>, использование тега <br> не добавляет пустой отступ перед строкой. Если текст, в котором используется перевод строки, обтекает плавающий элемент, то с помощью атрибута clear тега <b> можно сделать так, чтобы следующая строка начиналась ниже элемента.

Первая строчка


Вторая строчка
<p>Первая строчка</p><br><p>Вторая строчка</p>

center: выравнивает содержимое контейнера по центру относительно родительского элемента.

Текст в центре
<center>Текст в центре</center>

font: представляет собой контейнер для изменения характеристик шрифта, таких как размер, цвет и гарнитура.

Красный текст размера шесть гарнитуры Arial
<font color="red" face="Arial" size="6">Красный текст размера шесть гарнитуры Arial</font>;

ol и li: создаёт нумерованный список. Каждый элемент списка должен быть взят в тэг <li>.

    Нумерованный список:
  1. Раз
  2. Два
  3. Три
<ol>Нумерованный список:
<li>Раз</li>
<li>Два</li>
<li>Три</li>
</ol>

ul и li: создаёт маркированный список. Каждый элемент списка должен быть взят в тэг <li>.

    Маркированный список:
  • Красный
  • Зелёный
  • Синий
<ul>Маркированный список:
<li>Красный</li>
<li>Зелёный</li>
<li>Синий</li>
</ul>

img: предназначен для отображения на веб-странице изображений в графическом формате GIF, JPEG или PNG. Адрес файла с картинкой задаётся через атрибут src.

<img src="pic.jpg">

Таблица: table, tr и td

table: служит контейнером для элементов, определяющих содержимое таблицы.

tr: служит контейнером для создания строки таблицы. Каждая ячейка в пределах такой строки может задаваться с помощью тега <th> или <td>.

td: Предназначен для создания одной ячейки таблицы. Тег <td> должен размещаться внутри контейнера <tr>, который в свою очередь располагается внутри тега <table>.

Таблица
Заголовок Тоже заголовок
Данные Тоже данные

<table border="1">Таблица
<tr>
<th>Заголовок</th>
<th>Тоже заголовок</th>
</tr>
<tr>
<td>Данные</td>
<td>Тоже данные</td>
</tr>
</table>

iframe: Тег <iframe> создает плавающий фрейм, который находится внутри обычного документа, он позволяет загружать в область заданных размеров любые другие независимые документы. Тег <iframe> является контейнером, содержание которого игнорируется браузерами, не поддерживающими данный тег. Для таких браузеров можно указать альтернативный текст, который увидят пользователи. Он должен располагаться между элементами <iframe> и </iframe>.
<iframe src="index.html" width="100%" height="400" align="left">
Ваш браузер не поддерживает плавающие фреймы!
</iframe>
